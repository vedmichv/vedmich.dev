```
    в”ңв”Җв”Җ 2024-kubernetes-autoscaling.md

```

**One file per talk.** EN and RU files have matching slugs.

### Frontmatter Schema (Zod)

```ts
z.object({
  title: z.string(),                    // Talk title
  event: z.string(),                    // Conference/event name
  city: z.string(),                     // Event city
  date: z.coerce.date(),                // Event date
  tags: z.array(z.string()),            // Topic tags (lowercase)
  video: z.string().url().optional(),   // YouTube URL
  slides: z.string().url().optional(),  // Slidev URL (s.vedmich.dev/slug/)
  rating: z.string().optional(),        // Speaker rating e.g. "4.7/5.0"
  highlight: z.string().optional(),     // Highlight text e.g. "Speaker rating: 4.7/5.0"
  draft: z.boolean().default(false),
})
```

### Body Text

Markdown body text = talk description, key takeaways, notes. Optional вҖ” some talks may have empty body initially during migration.

## Interaction Patterns

### Homepage Speaking Section

| Interaction | Behavior |
|-------------|----------|
| Year text click | None вҖ” year is display text, not interactive |
| Event name click | None вҖ” event is display text, not interactive |
| Talk title click | Navigate to `/{locale}/speaking/{slug}` |
| `вҶ’` arrow | Part of talk text, not separate clickable element |
| "View all вҶ’" link | Navigate to `/{locale}/speaking/` (full portfolio page) |

**Scroll animation:** Each year group fades in via `.animate-on-scroll` (existing global CSS class, IntersectionObserver in `BaseLayout.astro`).

### Individual Talk Page

| Interaction | Behavior |
|-------------|----------|
| "Watch on YouTube" button | External link, `target="_blank"`, `rel="noopener noreferrer"`, opens YouTube video in new tab |
| "View Slides" button | External link, `target="_blank"`, `rel="noopener noreferrer"`, opens Slidev deck in new tab |
| Tag badge click | None вҖ” tags are display labels, not filters (no search/filter in v0.4) |
| "вҶҗ Back to Speaking" link | Navigate to `/{locale}/speaking/` |
| YouTube embed | iframe with `loading="lazy"`, aspect ratio 16:9, responsive width |

**Video embed implementation:** Use lite YouTube embed library OR native iframe with lazy loading. Executor chooses based on performance vs complexity trade-off. If lite-youtube, add NPM package and import. If iframe, use `loading="lazy"` attribute.

**Accessibility:**
- All external links have `aria-label` with " (opens in new tab)" suffix
- Video iframe has `title` attribute with talk title
- Tag badges have `role="list"` wrapper, `role="listitem"` per badge

## Data Migration Plan

### Source

`src/data/social.ts` вҖ” `speakingEvents` array (5 events, 6 talks total across 2026/2024/2023).

### Target

`src/content/speaking/{en,ru}/*.md` вҖ” one markdown file per talk with frontmatter.

### Migration Steps

1. Extract each talk from `speakingEvents` array
2. Create EN markdown file: `src/content/speaking/en/{YYYY-slug}.md`
3. Create RU markdown file: `src/content/speaking/ru/{YYYY-slug}.md` (mirror)
4. Populate frontmatter: title (talk), event (event.name), city (event.location), date (infer from year), tags (infer from title), video (add if known), slides (add if known), rating (event.highlight if present), draft (false)
5. Body text: empty initially (executor can expand later via QMD vault search)
6. After migration complete, remove `speakingEvents` export from `social.ts`
