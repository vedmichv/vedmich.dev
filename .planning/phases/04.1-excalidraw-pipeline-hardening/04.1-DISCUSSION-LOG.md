## Area 2a: White-rect strip implementation

| Option | Description | Selected |
|--------|-------------|----------|
| removeAttributesBySelector с narrow pattern (Recommended) | SVGO plugin with selector `path[fill="#fff"][d^="M0 0h"]` — strips only full-canvas white rect (d starts with `M0 0h<width>v<height>`). Narrow pattern won't break legitimate white shapes. Test: fixture with white rect + white shape verifies only the rect is removed. | ✓ |
| Custom visitor — strip first element if full-canvas white | Custom SVGO plugin inspects first `<path>` after `<svg>`, deletes if `fill=#fff` AND `d` covers full viewBox. More precise but ~30 LOC vs ~5. | |
| removeAttrs с pattern fill: #fff + d: M0 0h.*v | SVGO removeAttrs with regex pattern on fill and d. Simpler config but strips attributes without removing empty path node. | |

**User's choice:** removeAttributesBySelector с narrow pattern (Recommended)
**Notes:** Narrow selector beats full visitor — less code, equivalent precision. Test fixture design locks the "legitimate white shape survives" invariant.
