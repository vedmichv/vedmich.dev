### NEW — `src/lib/reading-time.ts` (D-27)

**Purpose:** Compute minute count from markdown body if frontmatter `reading_time` omitted. Shared between slug-page render and future skill (Wave 2).

**Signature:**
```ts
export function readingTimeMinutes(body: string): number {
  const wordCount = body.trim().split(/\s+/).length;
  return Math.ceil(wordCount / 200);
}
```

```ts
const readingTime = post.data.reading_time ?? readingTimeMinutes(post.body ?? '');
```
