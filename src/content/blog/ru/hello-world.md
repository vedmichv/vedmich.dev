# Hello World

Добро пожаловать на **vedmich.dev** — мой уголок интернета, где я делюсь тем, что узнаю, проектируя распределённые системы, работая с Kubernetes и создавая облачную архитектуру.

## Что будет

- **Заметки по архитектуре** — паттерны и анти-паттерны из реальных enterprise-проектов
- **Kubernetes глубоко** — от подготовки к CKA/CKS до дизайна production-кластеров
- **Расширенные версии докладов** — дополнительный контекст к выступлениям на re:Invent, AWS Summits и community-ивентах
- **Компаньон-посты к подкастам** — более глубокий технический контекст для DevOps Kitchen Talks и AWS на русском

## Почему персональный сайт?

После нескольких лет создания контента на YouTube, в подкастах и на конференциях, я хотел одно место, которое объединяет всё. Домашняя база, которая не принадлежит никакой платформе.

Сайт создан на [Astro](https://astro.build), стилизован Tailwind CSS и задеплоен на GitHub Pages. Ноль JavaScript на клиенте по умолчанию — именно так я люблю свою инфраструктуру: lean и эффективно.

Следите за обновлениями.
