# Slidev Presentations — UI Kit

Slide templates matching Viktor's Slidev decks at `s.vedmich.dev`. Dark theme, teal/amber, code-forward.

## Files
- `index.html` — runnable deck with 6 slide types (title, section, content, code, quote, thanks)
- `slides.jsx` — slide components

## Slide types
1. **Title** — brand gradient, big name, VV logo
2. **Section divider** — large overline + big heading
3. **Content** — title + bullets or 2-col
4. **Code focus** — dominant code block, sparse text
5. **Big quote** — pull quote, attribution
6. **Thanks** — CTA, social links
