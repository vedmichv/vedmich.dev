# vedmich.dev — Portfolio UI Kit

High-fidelity recreation of the personal portfolio at **vedmich.dev** — Astro 5 + Tailwind 4, dark theme only. Components are simplified for prototyping but match the live visual language.

## Files
- `index.html` — interactive click-thru of the full homepage (hero → about → podcasts → speaking → book → presentations → contact)
- `app.jsx` — all components, single file for brevity

## Source
`vedmichv/vedmich.dev` · data from `src/data/social.ts` and `src/i18n/en.json`.
